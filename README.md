# Big_Interger_class_for_C++
_last update: 21-09-08-00:50 GMT+9_

### 소개
동적 signed long 배열 이용, 2^k 진법 수로 저장  
배열의 i번 원소는 2^k^i 항의 계수  
k는 실험적으로 결정 (FFT 오차)
